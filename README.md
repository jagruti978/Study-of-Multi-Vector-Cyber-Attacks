# Study of Multi-Vector Cyber Attacks Using Steganography, SQL Injection, and Phishing

This project demonstrates practical cyberattack simulations using SQL Injection, Steganography, and Phishing in a controlled ethical environment. The study explains how attackers combine technical vulnerabilities and social engineering techniques to compromise systems and users.

The project includes:
- SQL Injection attack demonstrations on a dummy vulnerable website
- Steganography-based hidden payload transfer using Kali Linux and Steghide
- Phishing attack simulation using Zphisher on Kali Linux
- Detection and prevention techniques for each attack type
- SWOT analysis and cybersecurity countermeasures

## Tools & Technologies
- Kali Linux
- Steghide
- Zphisher
- SQL Injection Payloads
- Oracle VM VirtualBox

## Features Covered
- Multi-vector cyberattack simulation
- Payload hiding and extraction
- Authentication bypass demonstration
- Credential harvesting simulation
- Security detection and prevention mechanisms

## Disclaimer
This project was created strictly for educational and ethical cybersecurity learning purposes. All demonstrations were performed in controlled environments and dummy systems only.
